# Global Reach
В программе Global Reach демонстрируется работа с глобальными переменными.
Здесь будет показано, как обратиться к глобальной переменной из любой части
вашей программы. Здесь вы также узнаете, как скрыть глобальную переменную
в области видимости. Наконец, в этом программе мы научимся изменять глобальную
переменную из любой части программы.
